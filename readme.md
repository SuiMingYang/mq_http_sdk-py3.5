### 阿里云rocketmq sdk版本升级
[官方sdk地址](https://github.com/aliyunmq/mq-http-python-sdk)

阿里云rocketmq sdk目前只支持[2.5,3.0)，改写脚本部分代码，升级sdk，支持py3.0+开发。

拷贝mq_http_sdk文件夹到py安装环境的site-packages下即可。（可以溯源pandas或其他包找到site-packages路径）
